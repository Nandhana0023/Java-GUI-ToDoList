# Java-GUI-ToDoList
Author: Nandhana MA

Description: 
- A Java GUI-based To-Do List Application built using Swing.
Users can add tasks, view them in a list, and remove completed tasks.
- It demonstrates Object-Oriented Programming + GUI event handling in Java.
##  Technologies Used

- Java	

- Swing (javax.swing) - GUI components like JFrame, JButton, JList

- VS Code
## Features

✔ Add new tasks

✔ Display tasks in a scrollable list

✔ Remove selected tasks

✔ Simple GUI using Swing
## Contents Used

- Swing	- GUI toolkit for Java

- JFrame	- Main window

- JButton	- Buttons for actions

- JList + DefaultListModel	- Displays list items

- ActionListener	- Handles button clicks

- OOP + Event Driven Programming	- UI responds on events
